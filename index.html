    Java PMML API

    Project maintained by Thomas Badie, employee at Turn

    Welcome to JPMML Page.

    This is the homepage of the PMML project. It's an implementation in Java of the PMML language.

    It allows you to evaluate a model in PMML, or to generate a Java code that is equivalent to what does the model. I, @tbadie, am the current maintainer and I am working as a Turn employee.

    This project is intended to be open-source, which means that bug reports, questions, discussions and contributions are more than welcome.

    The wiki is currently under construction but will contain all the ressources about the project.

    Thank you for reading and hope you'll enjoy the project.
